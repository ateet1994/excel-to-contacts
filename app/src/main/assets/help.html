1.Creating Excel File

The application requires that the data be entered in the format shown in the image below.
The column format is as follows:
Name, Phone 1, Phone 2, Phone 3, Phone 4, Phone 5, Email 1, Email 2, Email 3, Email 4, Email 5.
The above order must be followed, otherwise the application would not be able store properly.

Zoom in to see clearly

The headings can be changed, but order of entry should be followed.

The Name and Phone 1 fields should not be empty. Phone 2 to Phone 5 and Email 1 to Email 5 fields can be empty.

2.Read From Excel

Click on '+' button and then click on Read From Excel, then choose an excel file. The file will be read and data will be displayed properly only if the order mentioned above is followed.

3.Read From Phone Book

Click on '+' button and then click on Read From Phone Book. The contacts read are only from default google account.

4.Write To Excel

Click on '+' button and then click on Write To Excel. The file will be stored in sdcard "ExcelToContacts/export.xls"

5.Write To Phone Book

Click on '+' button and then click on Write To Phone Book.

6.Backup Contacts From Phone Book

To backup, first read from phone book, then edit and delete if required, then write to excel.

7.Restore Contacts From Phone Book

To restore, first read from excel, then edit and delete if required, the write to phone book

8.Download from URL

An excel file can be downloaded from a server. To enable this a URL must be entered in settings
It should be in following format:
<protocol>://<web address>/<filename>.xls
Example:
"http://www.example.com/download.xls"
The file will be stored in sdcard "ExcelToContacts/download.xls"

9.Search

Click on search button, enter a query, then click enter on keyboard.
